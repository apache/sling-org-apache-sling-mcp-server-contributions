# Apache Sling MCP Server contributions

This project contains contributions ( tools, resources, prompts) for the Apache Sling MCP Server. See [the Sling MCP Server repository](https://github.com/apache/sling-org-apache-sling-mcp-server) for details on how to set up and run the MCP Server itself.
